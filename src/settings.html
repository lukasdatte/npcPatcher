Lukas' NPC Patcher

This patcher is meant to fix the Black face problem which is produced by overriding look Mods.
While it does that, it will search the whole npc for any changes to its' look. It will take the latest change that isn't identical to any base mod and merges it to any changes the lastest look mod did. That means it will make a NPC Overlaul (like i.e. Diversity or Bijinn) mod compatible with i.e. Immersive Weapons. But beware that when more non look mods edit one NPC, it will lake the data from the lastest.

Configure

But first you need to do some Configuring. If you press the Button down below, this tool will search all Mods that are editing humanoid NPCs.

  • Base

    Base Mods are mods or esp/ems files like Skyrim.esm or (UUESP) Unofficial Skyrim SE Patch. Those mods provide the base of the game. If you mark Skyrim a base mod, the patcher will know how it's NPCs look. Any mods that will make NPCs look the same, will not be flagged as look mods.

  • Look Mods

    Looks mods are mods that change an NPCs look compared to base mods. You can't configure look Mods sice that is don automatically on a NPC by NPC basis.

  • Copy Non Looks

    Non look Mods are Mods, that edit an NPC but don't change it's look. If you flag a Mod as "Copy Non look" the Patcher knows where to copy any changes made by the latest look mod. It will take the latest change by a look mod and merges it to any changes the lastest non look mod did.

  • Ignore

    The Patcher will simply ignore this Mod. You should ignore Bashed Patches and Patches Created by this Patcher.

  • Force No Patch

    If a NPC is edited by a Mod you flagged "Force No Patch" the Patcher will patch not patch this NPC.
